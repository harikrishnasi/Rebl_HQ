## Definition of shipped

Phase 3 is shipped when a founding house can run their night **without the founder
in the room**, using only the kit below.

## The kit

### Night format doc

One page: concept, run-of-show, staffing needs, what good looks like. Written per
house, from a shared template.

### Promo pack

- Poster + story templates in the venue's brand (not Rebl's).
- Caption pack: announcement, week-of, day-of, morning-after.
- QR code to the door list.

### Door list

Simplest thing that works: a form + a sheet the venue owns. Capture name and
return-visit consent. No app, no accounts, nothing to install.

## Explicitly not in Phase 3

- Any venue-facing dashboard or login.
- Automated messaging of any kind.
- Payments or ticketing — founding nights are free-entry by design.

## Measure

A night counts as working when ≥30% of door-list names from night N show up at
night N+1 without paid promotion.
